A Children quiz game that make them memorize the names of Electronic Objects which we are using in our daily life. It is a fun type learning game.

Features:
•	The series of quizzes about Electronic Objects which we are using in daily life.
•	Timer for the quiz which records the how much time the solving has taken.
•	Play/Pause and Stop/Restart Options.
•	Result in True or False is shown against every selection.
•	At the end Summary of How much questions were true, How much were False and how much time it took to solve the complete quiz.
•	Fully Different User Interface Designed keeping in focus point of view of Niche (Target) Market.
•	Live Tiles(Locally & Remotely)
•	Fully Unique Branding (Unique Logo Designing & Graphics Designing)
